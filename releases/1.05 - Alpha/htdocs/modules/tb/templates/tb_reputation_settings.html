Reputation System Settings
This section allows you to configure the User Reputation system.

Reputation On/Off
Enable User Reputation system?
Set this option to \'Yes\' if you want to enable the User Reputation system.
Yes   checked="checked"<{/if}> type="radio">    checked="checked"<{/if}> type="radio">   No
Default Reputation Level
Default Reputation
What reputations level shall new users receive upon registration? Make sure that you have a reputations level that is at least equal to or less than this value.
Default Reputation Phrase
If you have any user gain a reputations that exceeds your lowest negative level, then this phrase will be used for them. If you do not wish to use this phrase, make sure you set a negative reputations that is larger than the largest score (negative) that a user on your forum has.
Number of Reputation Ratings to Display
Controls how many ratings to display in the user's profile (userdeatails).
Reputation Powers
Administrator's Reputation Power
How many reputations points does an administrator give or take away with each click?
Set to 0 to have administrators follow the same rules as everyone else.
Register Date Factor
For every X number of days, users gain 1 point of reputation-altering power.
Post Count Factor
For every X number of posts, users gain 1 point of reputation-altering power.
Reputation Point Factor
For every X points of reputation, users gain 1 point of reputation-altering power.
User Reputation Settings
Minimum Post Count
How many posts must a user have before his reputations hits count on others?
Minimum Reputation Count
How much reputations must a user have before his reputations hits count on others?
Daily Reputation Clicks Limit
How many reputations clicks can a user give over each 24 hour period? Administrators are exempt from this limit.
Reputation User Spread
How many different users must you give reputations to before you can hit the same person again? Administrators are exempt from this limit.
